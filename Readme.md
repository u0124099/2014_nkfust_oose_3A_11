## 專題題目 : 左營咖啡館##


##一、動機 ##
       
### 這個系統是專門在介紹左營附近的所有咖啡館，因為在高雄這個地區，   左營的一些咖啡館蠻知名的，所以想要做這個系統方便咖啡愛好者快速找到咖啡館，並得到咖啡館的相關資訊，也可藉由這系統的排名資訊，去品嘗當月熱門咖啡館。
